Welcome!

You've reached your MHS home for the Hour of Code. The Hour of Code is a national event to help expose new people to computer programming and coding throughout the nation and the world. The MHS Byte Club wants to help you to learn to code!

  • It's not too hard.
  • It's not just for the super smart.
  • You don't need to be a math genious.
  • If you can use a computer - you can code!

Try your hand at today's programming activity

  • For Monday, November 30th: Build Flappy Bird in Scratch on Code.org
    • In this tutorial, you'll use a drag and drop editor to recreate the addictive mobile game Flappy Bird. You don't need any previous experience to be successful here. You can finish this whole project in homeroom!
    • Did you finish this tutorial? Brag about it here!
  • For Wednesday, December 1st: Learn to Draw with code at Khan Academy
    • Write your first lines of code to create drawings and animations using the JavaScript programming language
    • You can start this tutorial today, but and finish it with Byte Club on Tuesday at Byte Bash!

Byte Bash

Tuesday, December 8th, 3:30-7:00 PM in the District Meeting Room

  • Learn the basics of coding in a fun and friendly environment
  • Build something with your friends
  • Eat free food and get smarter in the process
  • Any student or teacher is invited to participate
